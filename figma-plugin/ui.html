How to use @buddy comments:

  1. Select any frame or element in your design
  2. Add a comment with "@buddy" and your question
  3. AI will analyze the visual design and reply
  4. Get instant feedback on colors, layout, spacing, etc.
Example comments:

@buddy Is this mobile-friendly?
@buddy Check the color contrast
@buddy General design review
